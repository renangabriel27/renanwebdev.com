Primeiro, atualize o sistema

```bash
sudo apt-get update
```

Depois, é preciso instalar o **build-essential package**.

```bash
sudo apt-get install build-essential
```

E o **curl**.

```bash
sudo apt-get install curl
```

Então, nós precisamos instalar o **rvm** mas primeiro instalar as GPG Keys.

```bash
gpg --keyserver hkp://keys.gnupg.net --recv-keys 409B6B1796C275462A1703113804BB82D39DC0E3 7D2BAF1CF37B13E2069D6956105BD0E739499BDB
```

Instalar RVM.

```bash
\curl -sSL https://get.rvm.io | bash -s stable
```

Depois da instalação rode o seguinte comando para o rvm funcionar.

```bash
source ~/.rvm/scripts/rvm
```

Também, para o RVM funcionar corretamente, você precisa marcar o *Run command as login shell in the terminal*.

### Comandos do RVM:

```bash
rvm  # show all commands of rvm
```

Instale o interpretador ruby.

```bash
rvm install 2.7.1 # add the last version stable of ruby
```

Para usar uma versão do ruby em específica.

```bash
rvm use 2.7.1
```

Para setar uma versão default do ruby.

```bash
rvm use 2.7.1 --default
```

Checar versão do ruby.

```bash
ruby -v
```

Instalar o rails.

```bash
gem install rails
```

Instale o Node Version Manager (NVM) para instalar o Node.js

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.3/install.sh | bash
ou
wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.35.3/install.sh | bash
```

Adicione o código abaixo do snippet em seu profile (~/.bash_profile, ~/.zshrc, ~/.profile, or ~/.bashrc).

```bash
export NVM_DIR="$([ -z "${XDG_CONFIG_HOME-}" ] && printf %s "${HOME}/.nvm" || printf %s "${XDG_CONFIG_HOME}/nvm")"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh" # This loads nvm
```

Instale o Node.js

```bash
nvm install --lts # install node.js lts version
```

Criando um novo projeto Rails

```bash
rails new app
```

  alt={`Rails new app`}

```bash
cd app/
rails s # Iniciar o server
```
